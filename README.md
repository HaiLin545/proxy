# proxy
server proxy script.

## Install

1. Add ```source /path/to/proxy/proxy.sh``` to your shell config. (etc, .bashrc, .zshrc)
2. ```source .bashrc```.

## Usage

1. ```set_proxy ip(optional) port(optional)``` default ip:port = 127.0.0.1:7890
2. ```unset_proxy``` 
3. ```test_setting```: test if your proxy is on
